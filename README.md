# CECA PAC 2015

### 目录结构
* `/paper`：保存paper的tex文件
* `/data`：测试中需要用到的全部源数据
* `/archive`：以往实现的版本，目前包含msbeam的cpu版本以及初赛的Xeon Phi优化版本
* `/docs`：存储需要用到的资料
* `/include`：代码用到的头文件
* `/src`：实现的`.cpp`文件
* `/build`：生成的可执行程序，这个本地维护就好
